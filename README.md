# Worship Motion Backgrounds

Self-contained WebGL worship motion backgrounds — a free alternative to paid loop packs.

- **16 styles:** Verlauf, Godrays, Rushing Waters, Stained Glass, Mosaic, Rings, Aurora, Refractions, Räucherwerk, Aufstieg, Kreise, Pixel Lines, Pixel Soft, Säulen, Mosaik Kreuz, JESUS, plus a two-photo **Foto-Mix**
- **13 palettes:** Dawn, Ember, Aurora, Deep, Gold, Champagner, Echtgold, Rose, Emerald, Royal, Crimson, Ocean, Ivory
- **Tempo slider** (static → fully dynamic) and a **Raster slider** for tile density on the grid-based styles
- Single file, no dependencies, runs in any modern browser

## Live

Open `index.html` — or via GitHub Pages: https://dkkrexa-tech.github.io/worship-backgrounds/

Controls: `[` / `]` switch style, keys `1`–`9` switch palette, `F` fullscreen, `T` sample lyric.
The **Mosaik Kreuz** and **JESUS** styles hide a shape in the mosaic that reveals itself through the motion (lit inverse to the background wave). **Foto-Mix** accepts two dropped photos and tiles them with random switching + palette-colour overlays.
